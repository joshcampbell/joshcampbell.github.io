Wikispex

Reading the Entrails of Wikipedia

This is an alpha release of Wikispex, a project aiming to create useful and interesting displays of data from Wikipedia. This preliminary version displays line-graphs showing trends in article size and user activity over time. Other charts and summaries are on their way.

Upcoming Features

  • Support for extremely active users and articles.
  • Summary pages with aggregate statistics.
  • Side-by-side charting of talk pages with their articles.
  • Analysis of article categories.
  • No more Twitter Bootstrap default theme.
